

# **Maven Market ‚Äì Power BI Sales Analytics Dashboard**

This repository contains a complete Power BI project built using the **Maven Market** dataset.
The report provides a detailed analysis of sales, customers, products, stores, and profitability across multiple regions. It demonstrates the full BI workflow ‚Äî data cleaning, modelling, DAX calculations, and interactive dashboard design.

---
 **Project Overview**

This Power BI solution was designed to help business teams understand:

* **Total Revenue, Profit & Margin**
* **Sales trends across months and years**
* **Top-performing stores and regions**
* **Product category & subcategory performance**
* **Customer behavior and segmentation**
* **Store-level breakdowns for performance comparison**

The dashboard is divided into multiple pages with easy navigation and dynamic slicers for deeper exploration.
